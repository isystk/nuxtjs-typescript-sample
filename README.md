## 操作コマンド

``` bash
# 環境構築
$ nodebrew install v12.13.0
$ nodebrew use v12.13.0

$ npm install -g yarn

$ yarn install

# ビルド&起動
$ yarn run dev
http://localhost:3000/

# ビルドだけ、起動だけ
$ yarn run build
$ yarn start

# 成果物の生成
$ yarn run generate
```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
