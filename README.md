Nuxt.js TypeScript Vuetify サンプル
====

Nuxt.js、TypeScript、Vuetify を利用したサンプルです。

## Description

Nuxt.js、TypeScript、Vuetify を学習する目的で、自分のポートフィリオを作成してみました。

## VS. 

## Requirement

* Nuxt.js@2.10.2
* Vue.js@2.6.10 
* Vuetify@2.1.6 

## Usage

``` bash
# ビルド&起動
$ yarn run dev
http://localhost:3000/

# ビルドだけ、起動だけ
$ yarn run build
$ yarn start

# 成果物の生成
$ yarn run generate
```

## Install

``` bash
# 環境構築
$ nodebrew install v12.13.0
$ nodebrew use v12.13.0

$ npm install -g yarn

$ yarn install
```
