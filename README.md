## Install

``` bash
# 環境構築
$ nodebrew install v12.13.0
$ nodebrew use v12.13.0

# yarn(パッケージマネージャ)のインストール
$ npm install -g yarn

# 依存モジュールのインストール
$ yarn install

# ビルド&起動
$ yarn run dev
$ curl http://localhost:3000/
```
